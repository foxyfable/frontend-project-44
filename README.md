###How to install
```
make install
```

###How to run a game
**Is the number even?**
```
brain-even
```

**Calculate the equation**
```
brain-calc
```

**Find greatest common divisor**
```
brain-gcd
```

**Is the number prime?**
```
brain-prime
```

**Complete the progression**
```
brain-progression
```
